## Python Security Utilities

This repository includes small Python utilities for authorised lab work and defensive analysis.

| Script | Purpose | Example |
| --- | --- | --- |
| `Scripts/Python/port_scanner.py` | Scan TCP ports on an authorised target | `python3 Scripts/Python/port_scanner.py 127.0.0.1 -p common` |
| `Scripts/Python/cve_lookup.py` | Search CVEs using the NVD API | `python3 Scripts/Python/cve_lookup.py --keyword apache --limit 3` |
| `Scripts/Python/password_checker.py` | Check password strength offline | `python3 Scripts/Python/password_checker.py` |
| `Scripts/Python/log_analyzer.py` | Summarise web/server logs | `python3 Scripts/Python/log_analyzer.py access.log` |
| `Scripts/Python/ioc_scanner.py` | Extract IPs, domains, URLs, hashes, and emails from files | `python3 Scripts/Python/ioc_scanner.py evidence.txt` |
| `Scripts/Python/whois_lookup.py` | Look up WHOIS or RDAP registration data | `python3 Scripts/Python/whois_lookup.py example.com --rdap` |
| `Scripts/Python/subdomain_enum.py` | Resolve likely subdomains for an authorised domain | `python3 Scripts/Python/subdomain_enum.py example.com` |
